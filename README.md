# Macro Tracker App

A JavaFX desktop application for tracking food intake and daily macronutrients.

## Features
- Add food entries
- Delete entries
- Live calorie and macro totals
- Save/load data using JSON

## Tech Stack
- Java
- JavaFX
- Gson

## How to Run
1. Clone the repository
2. Open in IntelliJ
3. Add JavaFX and Gson dependencies
4. Run MacroTrackerApp.java
